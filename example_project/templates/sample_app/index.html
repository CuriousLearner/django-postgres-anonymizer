Quick Start Guide
1. Initialize Extension
python manage.py anon_init
2. Create Rules
python manage.py anon_load_yaml presets/django_auth.yaml
3. Apply Anonymization
python manage.py anon_apply
4. Export Anonymized Data
python manage.py anon_dump output.sql
